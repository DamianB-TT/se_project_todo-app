# Simple Todo App

A simple to-do list web app that allows users to add, remove, and manage tasks.

## Functionality

Users can:

- Add new tasks to the list
- Mark tasks as completed or not completed
- Remove tasks from the list

The app uses form validation to ensure inputs are handled correctly and provides a smooth user experience when interacting with tasks.

## Technology

This project is built using:

- HTML
- CSS
- JavaScript (Event handling, form validation, modules)
- BEM Methodology
